Welcome to GitHub Pages.

このページは、github上のコンテンツページです。 KD学生(2年生、Java/DB)は「リリース版」側のカタログをEclipseもしくはMavenで使ってください。 なお、将来的には両者のカタログを統合する予定です。

  • リリース版アーキタイプカタログ(XML形式)
  • スナップショット版アーキタイプカタログ(XML形式)

注意: kdwebをEclipse 3.4のAll in Oneバージョン(Pleiades - Eclipse プラグイン日本語化プラグインにて配布されるもの)で利用した場合、 testゴールは動作するようですが、integration-testゴールを実行しようとすると、Eclipse実行用のJavaVMバージョンの都合と思われる障害で処理中にエラーを起こすことがあります。 手元では、3.7のAiOにて動作チェックを行っております。4.2のAiOの動作は未確認です。
